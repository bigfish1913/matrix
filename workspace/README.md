# Tank Tower Defense

A tower defense game built with Rust and Bevy.

## Requirements

- Rust 1.80 or later
- Cargo

## Running the Game

```bash
cargo run
```

## Project Structure

```
tank-tower-defense/
├── Cargo.toml          # Project configuration and dependencies
├── src/
│   └── main.rs         # Game entry point and basic setup
├── assets/
│   ├── sprites/        # Game sprites and textures
│   ├── sounds/         # Sound effects and music
│   └── fonts/          # Custom fonts
└── README.md           # This file
```
